VMG Viewer — простая кроссплатформенная утилита на Python для просмотра SMS, экспортированных в формате .vmg (Samsung). Программа:

Читает .vmg файлы в указанной папке (по умолчанию — папка скрипта).
Декодирует QUOTED-PRINTABLE и поддерживает UTF-8/CP1251/Latin-1.
Извлекает номер контакта, дату и текст сообщения (поддержка folded lines).
Предоставляет удобный GUI (tkinter) с выбором папки, списком контактов и просмотром переписок.
Почему этот репозиторий полезен:

Лёгкий, зависит только от стандартной библиотеки Python.
Практичен для резервного просмотра и анализа SMS из старых телефонов.
Код прост для расширения (экспорт в CSV, поиск, фильтры).
Краткие инструкции:

Клонировать репозиторий.
Запустить: python3 vmg_viewer.py
Нажать "Выбрать папку с sms" и указать папку с .vmg файлами.
