# vmg-viewer
VMG Viewer — утилита на Python для просмотра SMS, экспортированных в формате .vmg (Samsung). Читает .vmg файлы в указанной папке (по умолчанию — папка скрипта). Декодирует QUOTED-PRINTABLE и поддерживает UTF-8/CP1251/Latin-1. Извлекает номер контакта, дату и текст сообщения (поддержка folded lines). Предоставляет удобный GUI (tkinter).
